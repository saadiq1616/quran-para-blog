#🌙 Quran Ke 30 Para – Hinglish Blog

> Har Para Ki Kahani — Hinglish Mein, Seedhi Dil Tak

## Kya Hai Yeh Site?

Quran ke tamam 30 Para (Juz) ki kahaniyan — easy Hinglish mein likhi gayi hain 
taaki har koi samajh sake. Chahe aap pehli baar Quran parh rahe ho ya purana 
rishta taza karna chahte ho — yeh blog aapke liye hai.

## Features

- 📖 **30 Para Blog** — Har para ki mukammal kahani
- 👤 **Kirdar** — Kaun kaun hain har para mein
- 🕰️ **Zamana** — Kab aur kahan hua
- 💬 **Paigham** — Kya seekhna chahiye
- 🚫 **Myths vs Haqeeqat** — Galat fehmiyan door karo
- 🤖 **AI Q&A** — Koi bhi sawaal poochho, Quran se jawab milega
- 🔍 **Smart Search** — Para, topic ya sawaal — sab dhundho

## Kaise Use Karein?

1. **Para Blog** — Kisi bhi para card pe click karo, poori kahani padho
2. **Search** — Upar search box mein topic ya sawaal likho
3. **AI Sawaal** — "Sawaal Poochho" button dabao, koi bhi sawaal karo

## Tech

- Pure HTML, CSS, JavaScript — koi framework nahi
- Anthropic Claude API for AI Q&A
- Hosted on GitHub Pages — free forever
